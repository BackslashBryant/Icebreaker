# Handoff Command (target a persona)

Drop this to construct a crisp handoff to a specific teammate.

```
@<Persona Name + Emoji> Plan Mode only.

Context:
- Plan checkpoint: <n>: <title>
- Files: <precise paths>
- Tests: <exact commands + expectations>
- Constraints: <versions, API contracts, perf/accessibility gates>

Deliver:
- Status: how you’ll approach this checkpoint
- Next 3: files you’ll touch, tests you’ll run, rollback plan
- Question: anything you need clarified (or Question: none)

Wait for “Proceed with checkpoint <n>” before editing. After each change, run the targeted test and paste output.
```
