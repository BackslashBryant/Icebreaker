# Icebreaker Alpha Test

**No installation needed!**

## Quick Start (2 Steps)

1. **Open your browser** (Chrome, Firefox, or Edge recommended)
2. **Visit**: https://frontend-coral-two-84.vercel.app

**That's it!** Start testing immediately.

---

## What to Test

### Onboarding Flow
- Click "PRESS START"
- Complete the 4 steps:
  - What We Are/Not
  - 18+ Consent
  - Location (allow browser permission)
  - Choose your vibe and tags
- See your generated handle

### Radar View
- View nearby people (if others are testing)
- Toggle between CRT Sweep and List views
- Click on people to see their details

### Chat
- Click "START CHAT" on someone's card
- If they accept, chat in real-time
- Messages are ephemeral (disappear when chat ends)

### Safety Features
- **Panic Button**: Always visible (bottom-right corner)
- **Block/Report**: Right-click or long-press on a person's card

---

## Testing with a Friend

**Both of you:**
1. Open the same URL on your laptops/devices
2. Complete onboarding
3. You'll see each other on Radar (if you're physically close)
4. Try starting a chat!

**Note**: Location services work best when you're actually near each other. If testing remotely, you may not appear on each other's Radar.

## Troubleshooting

**Page won't load?**
- Check your internet connection
- Try a different browser
- Clear browser cache and try again

**Location not working?**
- Make sure you allowed location permission
- HTTPS is required for location (already enabled on production)

**Can't see other people?**
- Make sure both testers completed onboarding
- Check that visibility is ON (Profile settings)
- You need to be physically close for proximity matching

**Chat not working?**
- Ensure both users are online
- Check browser console for errors (F12 → Console)
- Try refreshing the page
